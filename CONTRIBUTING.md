# Contributing to GEO-SEO opencode

First off, thank you for considering contributing! This project is an opencode adaptation of [geo-seo-claude](https://github.com/zubair-trabzada/geo-seo-claude) by Zubair Trabzada.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, search the Issues first. If no existing issue matches, open a new one with:
- A clear title and description
- Steps to reproduce
- Your opencode version and OS

### Suggesting Enhancements
Open a feature request under Issues with a clear explanation of the feature and why it matters.

### Pull Requests
1. Follow the styleguides below
2. Ensure compatibility with both the GEO methodology and opencode's architecture
3. Verify that all status checks pass

## Styleguides

### Git Commit Messages
- Use present tense ("Add feature" not "Added feature")
- Use imperative mood ("Move cursor to..." not "Moves cursor to...")
- Limit first line to 72 characters
- Reference issues and PRs liberally

### Documentation
When modifying skills or behavior, update the corresponding SKILL.md files and README.md as needed.
